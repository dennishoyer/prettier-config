# @dennishoyer/prettier-config

## Installation

Install the package via npm.

```bash
pnpm install -D @dennishoyer/prettier-config
```

## Implementation

Add the prettier line into your package.json.

```json
{
  "name": "@author/package",
  "scripts": { },
  "devDependencies": { },
  "prettier": "@dennishoyer/prettier-config"
}

```
